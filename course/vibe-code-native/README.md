# Vibe Code Native — Build Real iOS Apps with AI

## Course Overview

A comprehensive, hands-on course that teaches iOS developers how to leverage AI tools — especially Claude Code — to build production-quality native apps faster, smarter, and with fewer bugs.

**Target Audience:** iOS developers (junior to mid-level) who want to integrate AI into their native development workflow.

**Prerequisites:**
- Basic Swift knowledge (variables, functions, structs, classes)
- Xcode installed (16.0+)
- Familiarity with SwiftUI (basic views, navigation)
- No AI experience required

**Total Duration:** ~8-10 hours of video content across 7 modules

## Script Format

Each lesson script (`.md` file) follows a consistent structure:

1. **Meta** — Duration, video format, required resources
2. **Outline** — Bullet-point overview for quick reference
3. **Full Script** — Word-for-word narration text (what Mario says on camera)
4. **Screen Notes** — `[SCREEN: description]` markers for what appears on screen
5. **Code Snippets** — All code ready for live demos
6. **Key Takeaways** — End-of-lesson summary
7. **Homework** — Optional exercise for students

## Production Notes

### Recording Setup
- **Screen Recording:** OBS (free) or ScreenFlow ($169, macOS)
- **Code Editor:** Xcode 16+ with default dark theme
- **Terminal:** Claude Code running in iTerm2 or Terminal.app
- **Resolution:** 1920x1080 minimum, 2560x1440 recommended
- **Audio:** External USB microphone, quiet room

### Editing Checklist
- [ ] Code on screen is readable (font size 16+ in Xcode)
- [ ] Terminal text is visible (font size 14+)
- [ ] No sensitive data visible (API keys, personal info)
- [ ] Smooth transitions between screen regions
- [ ] Key moments have text overlays for emphasis

### Brand Guidelines
- Tone: Direct, conversational, data-driven, opinion-forward
- Always explain *why*, not just *what*
- Acknowledge tradeoffs honestly
- Native-first philosophy throughout
- Privacy emphasis where relevant
